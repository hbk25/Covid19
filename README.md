# COVID-19 Detection using CNN

## Overview
This project uses a Convolutional Neural Network (CNN) built with TensorFlow to detect COVID-19 from image data.  
The notebook covers the full pipeline, from preprocessing images to training and evaluating the model.

---

## Workflow

### 1. Image Processing
- Load and resize images  
- Normalize pixel values  
- Prepare training and validation datasets  

### 2. Model
- CNN built using TensorFlow/Keras  
- Convolution and pooling layers  
- Dense layers for classification  

### 3. Training & Evaluation
- Train the model on labeled data  
- Track accuracy and loss  
- Evaluate performance on validation data  

---

## Tech Stack
- Python  
- TensorFlow / Keras  
- NumPy  
- Matplotlib
